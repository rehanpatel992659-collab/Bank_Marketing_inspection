# Bank_Marketing_inspection
# Bank Marketing Campaign Analysis & Customer Subscription Prediction

## Project Overview

This project analyzes a bank marketing dataset and predicts whether a customer will subscribe to a term deposit using Machine Learning techniques.

The project includes data cleaning, exploratory data analysis (EDA), feature engineering, class balancing using SMOTE, and predictive modeling with multiple machine learning algorithms.

## Objectives

* Analyze customer behavior and marketing campaign performance.
* Identify factors influencing term deposit subscriptions.
* Build machine learning models for customer subscription prediction.
* Generate actionable business insights.

## Dataset Information

* Total Records: 41,188
* Features: 21
* Target Variable: Customer Subscription (Yes/No)

## Machine Learning Models

* Logistic Regression
* Random Forest Classifier
* XGBoost Classifier

## Results

* Best Model: Random Forest
* Accuracy: ~91%
* Key Predictive Features:

  * Duration
  * Euribor3m
  * Age
  * Campaign
  * Previous Campaign Outcome

## Business Insights

* Call duration is the strongest predictor of subscription.
* Previous successful campaign responses increase conversion probability.
* Economic indicators significantly influence customer decisions.
* Targeted marketing can improve campaign effectiveness.
